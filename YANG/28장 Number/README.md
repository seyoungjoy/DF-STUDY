# Number
## Number 프로퍼티
### Number.MAX_VALUE
- js에서 표현할 수 있는 가장 큰 양수
### Number.MIN_VALUE
- js에서 표현할 수 있는 가장 작은 양수

## Number 메서드
### isFinite
- Infinity/-Infinity 검사
### isInteger
- 정수 boolean 값 반환
### isNaN
- NaN 검사 후 boolean 값 반환
### isSafeInteger
- 안전한 정수인지 검사
### toExponential
- 숫자를 지수 표기법으로 반환
### toFixed
- 숫자를 반올림하여 문자열로 반환
### toPrecision
- 인수로 전달받은 자릿수까지 유효하도록 반올림
### toString
- 숫자를 문자열로 변환
