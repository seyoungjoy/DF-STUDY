# 18장 함수와 일급 객체
## 18.1 일급 객체
- 함수는 일급객체이기 때문에 객체와 동일하게 사용할 수 있다.
- 하지만 일반객체는 호출 x, 함수는 호출할 수 있으며
- 함수 객체는 일반객체에는 없는 함수 고유의 프로퍼티를 소유한다.
